# craSH's dotfiles

My personal dotfiles managed using [chezmoi](https://www.chezmoi.io/)

## 🚀 Install

1) [Install chezmoi](https://www.chezmoi.io/install/)
2) Install dotfiles:

    ```console
    chezmoi init --apply https://github.com/crash/dotfiles.git
    ```
